# ClothImage - Bibliothèque de Vêtements


Ce projet contient une bibliothèque d'images de vêtements provenant du pack de S4na Dev :

```
ClothImage/
├── femme/          # Assets pour personnages féminins
│   ├── arms/       # Manches/Bras
│   ├── bags/       # Sacs
│   ├── bodyarmors/ # Gilets pare-balles
│   ├── bracelets/  # Bracelets
│   ├── ears/       # Accessoires d'oreilles
│   ├── glasses/    # Lunettes
│   ├── hats/       # Chapeaux
│   ├── masks/      # Masques
│   ├── pants_1/    # Pantalons
│   ├── shoes_1/    # Chaussures
│   ├── torso_1/    # Hauts de corps
│   ├── tshirt_1/   # T-shirts
│   ├── undershirts/# Sous-vêtements
│   └── watches/    # Montres
│
└── homme/          # Assets pour personnages masculins
    ├── arms/       # Manches/Bras
    ├── bags/       # Sacs
    ├── bodyarmors/ # Gilets pare-balles
    ├── bracelets/  # Bracelets
    ├── ears/       # Accessoires d'oreilles
    ├── glasses/    # Lunettes
    ├── hats/       # Chapeaux
    ├── masks/      # Masques
    ├── pants_1/    # Pantalons
    ├── shoes_1/    # Chaussures
    ├── torso_1/    # Hauts de corps
    ├── tshirt_1/   # T-shirts
    ├── undershirts/# Sous-vêtements
    └── watches/    # Montres
```

## 🎯 Utilisation

Ces assets peuvent être intégrés dans des scripts de magasins de vêtements / barber ou autres projets similaires, permettant d'associer une image au bon assets.

example : bags 8 = 8.png du dossier bags à vous d'adapter votre projet au github 

vous pouvez utilisez cela : https://raw.githubusercontent.com/xBackend/ClothImages/main


## 📝 Format des Fichiers

- **Format** : PNG
- **Nommage** : Numérique séquentiel (0.png, 1.png, 2.png, ...)
- **Organisation** : Par genre puis par catégorie


- Toutes les images sont au format PNG pour supporter la transparence
